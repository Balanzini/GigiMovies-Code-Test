# GigiMovies Code Test

This app shows a list of popular movies, where you can select your favourites to create your personalised list.

### Built with:

* **Programming language:** [Kotlin]
* **Architectural pattern:** [MVVM]
    * With [ViewModel], [Flow] and [LiveData]
* **DataBase:** [Room]
* **Dependency Injection:** [Koin]
* **Http Client:** [Retrofit2]
* **Image loader:** [Glide]
* **Implementation of navigation:** [Navigation component]
* **Concurrency design pattern:** [Coroutines]

## MovieDB API
The application uses [MovieDB Api] to fetch the data. You have to put your own api key in com.jose.gigimovies.data.retrofit.MovieRetrofitSource to be able to test the app.
